Lumina SaaS Platform

Client

Lumina FinTech

Role

Lead Frontend Developer

Timeline

12 Weeks

Services

UI Engineering, Motion Design

The Challenge

Lumina approached me with a complex problem: their financial dashboard was incredibly powerful, but overwhelming for new users. The goal was to redesign the frontend architecture and UI to feel less like a spreadsheet and more like a premium luxury consumer app.

I had to balance high-density data visualization with large amounts of whitespace, utilizing micro-interactions to reveal secondary information only when needed.

The Approach

We started by rebuilding the component library from the ground up using React and Tailwind CSS. To achieve the "premium" feel, we implemented custom GSAP animations for all data state changes.

Instead of harsh grid lines, we used subtle shadows, floating cards, and glassmorphism effects to create a sense of depth and hierarchy.
